# random-items
Docker container setup for random items Minecraft minigame.

## Installation
- Clone/download this repository.
- Install Docker Desktop/Docker Engine.
- Open terminal in the project directory and run `docker compose up -d`.

### To start already created server use `docker start random-items-mc-1`.

## This project contains following datapack:
[Random Item Giver Datapack \[Minecraft 1.16.5 - 1.21.11\]](https://www.planetminecraft.com/data-pack/random-item-giver-datapack-1-0-0-minecraft-1-16-2/)
